Number Click Game

The goal is to click all the numbers in order as fast as you can. The game board is a square grid.

For example: Easy - 4x4 grid with 4 cells on each side.

Select a difficulty level and click the Start Game button to begin. Click the Cancel Game button to cancel the game.
